Join us Thursday Feb 13, 2014 as Jeff Galloway from HP talks about how HP is extending your PowerShell capabilities to enable core HP ProLiant server  features. HP Scripting Tools for Windows PowerShell is a powerful set of utilities that can be used to perform various configuration tasks on HP  
hardware.  
Check it out at [www.hp.com/go/powershell](http://www.hp.com/go/powershell) and click on the Technical Support link to get to the download page, then click on Software, Drivers, and Firmware. There is also a User Guide if you click “Manuals” on the left side of the page
